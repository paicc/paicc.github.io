投資基本觀念

Posted on 2018-11-22 | | Visitors:


這星期參加VV舉辦的小資投資心法，在這邊做點筆記紀錄一下

首先推薦一下VV粉絲團，會告知投資新知和重點新聞訊息，小資參考一下滿不錯的。

投資工具：

首先先介紹一下投資工具，請記住保險並不應該視為同資工具，保險比較頃向用於意外的保障，投資工具分為以下四類：

1. 股票

  • 需要隨時關注市場動態
  • 需要能看懂公司財報做操作

    2. 共同基金

  • 由基金經理人主動選股
  • 可由操盤人增減持股
  • 買基金前準備功課
    1. 選定基金種類（原物料，國家，債券等）
    2. 翻閱基金月報，了解該基金持股，投資意向及績效表現等等

      以中國基金為例，貝萊德中國基金， 摩根中國基金

    3. 查詢3年 1年 半年的該基金績效是否能超越指數，如果沒有任何基金績效超越指數會挑選被動型基金

3. ETF(指數股票型基金)

  • 貼近指數，但不會有超額報酬
    -

    4. 黃金

  • 黃金投資方法為低買高賣，主要目的為抵禦風險之用，一般不會納入投資組合中。

5. 房地產

介入投資標的方法(重要性由高到低)

1. 基本面分析

  • 國家財務狀況:GDP(國家生產總值), PMI(採購經理人指數)等等數據。

    2. 技術線圖分析

  • 做為基本面的輔助

    3. 籌碼面分析

  • 資金流入流出

    4. 消息面分析

  • 新聞，評論等等（需自行判斷合理性及邏輯性）

影響標的漲跌因素

1. 經濟發展前景

主要觀察數據
1. PMI(製造業＆服務業採購經理人指數)
    - 50以上為擴張，50以下為委縮
    - 美國稱作ISM
2. CPI(消費者物價指數)
3. GDP(經濟成長率)
4. 其他(失業率、經濟體主要依賴數據)

2. 政治風險

3. 重大國際事件(脫歐公投，貿易戰)

4. 不可抗拒因素(自然災害，恐怖攻擊)
